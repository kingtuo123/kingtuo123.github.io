创建SSH Key#

运行下列命令创建SSH Key，邮箱使用github创建时用的邮箱。

ssh-keygen -t rsa -C "kingtuo123@foxmail.com"

输出如下，一路回车

Generating public/private rsa key pair.
Enter file in which to save the key (/home/king/.ssh/id_rsa): 
Enter passphrase (empty for no passphrase): 
Enter same passphrase again:

以上会在 ～/.ssh 目录下生成 id_rsa （私钥）和 id_rsa.pub （公钥）文件。

如果你没有使用默认的路径，记得修改 /etc/ssh/ssh_config 或 ~/.ssh/config

#   这里是默认身份文件路径
#   IdentityFile ~/.ssh/id_rsa
#   IdentityFile ~/.ssh/id_dsa
#   IdentityFile ~/.ssh/id_ecdsa
#   IdentityFile ~/.ssh/id_ed25519

添加公钥#

查看公钥的内容

cat ~/.ssh/id_rsa.pub

打开github在 setting -> SSH and GPG keys 中添加SSH Key，将上面的输出复制进去。

验证#

ssh -T git@github.com

返回如下信息则配置成功

You've successfully authenticated, but GitHub does not provide shell access
