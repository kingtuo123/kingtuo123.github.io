寄存器#

STK_CTRL#

  • SysTick控制和状态寄存器（ SysTick control and status register ）
  • COUNTFLAG: 如果在上次读取本寄存器后， SysTick 已经计到了 0，则该位为 1
  • CLKSOURCE: 时钟源选择
    • 0：AHB/8
    • 1：处理器时钟 AHB
  • TICKINT: SysTick异常请求使能
    • 0：数到 0 时 无动作。也可以通过读取 COUNTFLAG 标志位来确定计数器是否递减到0
    • 1：SysTick 倒数计数到 0 时产生 SysTick 异常请求
  • ENABLE: SysTick 定时器的使能位
    • 0：禁止
    • 1：使能

STK_LOAD#

  • 重装载数值寄存器（SysTick reload value register）
  • RELOAD[23:0]: 当倒数计数至零时，将被重装载的值

STK_VAL#

  • 当前数值寄存器（SysTick current value register）

CURRENT[23:0]: 读取时返回当前倒计数的值，写任意值则使之清零（包括COUNTFLAG）

STK_CALIB#

  • 校准数值寄存器（SysTick calibration value register）
