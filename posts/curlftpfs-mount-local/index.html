Curlftpfs挂载ftp到本地

March 9, 2021
Table of Contents
  • 内核选项
  • 安装
  • 以普通用户挂载
  • 卸载
  • 参考 CurlFtpFS

内核选项#

File systems --->
   <*> FUSE (Filesystem in Userspace) support

安装#

emerge -av net-fs/curlftpfs

以普通用户挂载#

首先，创建挂载点

mkdir ./ftp

挂载

curlftpfs ftp://server/catalog/ ./ftp/ -o user=username:password,utf8,ssl

ssl ：使用 SSL/TLS 传输数据

utf8 ： 使用 utf8 编码

卸载#

fusermount -u ~/example

或者

umount ~/example
